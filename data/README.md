# data/

This folder holds all data used by the Iqraa AI pipeline.

## Folder Structure

```
data/
├── text/           # Canonical Qaloon reference text (committed to git)
│   └── qalon_canonical.json
├── audio/          # Raw surah MP3 files (GITIGNORED — never commit)
└── processed/      # Generated by notebooks — created at runtime
    ├── segments/   # Ayah-level 16 kHz WAV files
    ├── labels.json # Ayah ID → diacritized text
    ├── vocab.json  # CTC character vocabulary
    └── hf_dataset/ # HuggingFace Dataset directory
```

## What Goes Where

### `data/text/`
Place `qalon_canonical.json` here. This file is the King Fahd Glorious Quran
Printing Complex (KFGQPC) edition of the Quran as recited in Riwayat Qaloon.
It contains 6 214 fully diacritized ayahs and is safe to commit to git.

Schema:
```json
[
  { "surah": 1, "ayah": 1, "text": "بِسْمِ اللَّهِ الرَّحْمَٰنِ الرَّحِيمِ" },
  ...
]
```

### `data/audio/`
Place the Ali al-Hudhaifi (Qaloon) surah-level MP3 files here. Name them
`001.mp3` through `114.mp3` (zero-padded surah number).

**This folder is in .gitignore. Do NOT commit audio files.**  
Audio files are large binaries — they belong on Google Drive or a dedicated
storage bucket, not in version control.

### `data/processed/`
Created automatically by the notebooks. Do not create this folder manually.
Contents are also gitignored (generated artifacts).
